Pet Finder Website Details

As part of the Codubee Remote internship program, I contributed to the PetFinder web application, a platform designed to connect shelter animals with people looking to adopt. I developed reusable React components to build the user interface, including dynamic listing pages and interactive adoption features. Working within an Agile team, I presented progress at biweekly standups and sprint reviews, helping the team iterate on feedback and plan deliverables. This project emphasized component-based design and responsive UI development.

The application was structured with a dedicated front end in React and supported by a backend server, enabling end‑to‑end functionality for listing and browsing pet profiles. I implemented features such as pet listings and user interactions that improve the adoption experience. Using Jira, we organized work and maintained clear documentation of our tasks, ensuring we met sprint goals and delivered software on schedule. My role also involved integrating collaborative feedback and refining components based on usability insights.

Lesli just completed her first React component and successfully merged it into the main branch! She and her team are working on a solution that pairs shelter animals with people looking to adopt.
